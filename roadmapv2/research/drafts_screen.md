# Drafts Screen - Nature Distilled

## Purpose

- Show in-progress items that need edits.

## Layout

```
GlassAppBar: Drafts
List
  - Draft rows (compact cards)
Floating CTA
  - New scan / continue scanning
```

## Tokens (Color + Typography)

Use v2 ramps from `roadmapv2/research/color_palette_tokens.md` and text styles from `roadmapv2/research/typography_tokens.md`.

Color:
- background: neutral0
- draft status badge: mutedTeal or sage (non-destructive)

Typography:
- title: displayMedium
- row title: titleMedium
- helper: bodySmall

## Empty State

- Use paper texture + the coffee cup empty state widget.
