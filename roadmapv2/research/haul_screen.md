# Haul Screen - Nature Distilled

## Purpose

- Group items scanned in a session (haul) and show quick totals.

## Layout

```
GlassAppBar: Haul
Hero summary card
  - Items count
  - Total spend
  - Estimated resale
  - Estimated profit
List/Grid
  - Item cards
```

## Tokens (Color + Typography)

Use v2 ramps from `roadmapv2/research/color_palette_tokens.md` and text styles from `roadmapv2/research/typography_tokens.md`.

Color:
- background: neutral0
- summary accents: terracotta500 for highlights, sage for profit

Typography:
- summary numbers: statLarge/statMedium
- labels: labelMedium
- list titles: titleMedium
