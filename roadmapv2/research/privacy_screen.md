# Privacy Screen - Nature Distilled

## Purpose

- Provide human-readable privacy controls and explanations.
- Avoid heavy glass for long text; prioritize readability.

## Layout

```
AppBar: Privacy
Scroll
  - Summary card (what we store)
  - Toggles (send crops, comps)
  - Links (policy)
```

## Tokens (Color + Typography)

Use v2 ramps from `roadmapv2/research/color_palette_tokens.md` and text styles from `roadmapv2/research/typography_tokens.md`.

Color:
- background: neutral0
- content surface: neutral2 (mostly opaque)
- text: ink0

Typography:
- title: displayMedium
- body: bodyMedium (line height comfortable)
- helper text: bodySmall

## Asset Suggestions (roadmapv2/images)

- Background: none or `roadmapv2/images/loppis_background.png` heavily tinted/blurred
